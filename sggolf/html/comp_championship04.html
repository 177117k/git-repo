신페리오 방식이란 ?

18홀 중 파의 합이 48이 되도록 임의로 12홀을 숨김홀로 구성하여, 경기 종료 후 숨김홀 스코어 합에 1.5배한 후 해당 코소의 총 Par(72)를 뺀 값의 80%를 핸디캡으로 하는 방식입니다.

신페리오 스코어 계산법 예

  • { ( 숨긴홀 스코어 합계 x 1.5 ) – 72(해당 코스 총 Par 수) } X 0.8 = 신페리오 핸디
  • 스트로크 스코어 – 신 페리오 핸디 = 신페리오 스코어

스트로크 스코어(A) 집계방식

스트로크 스코어(A)는 대회에 참가한 전체 라운드 중 신페리오 스코어가 가장 좋은 라운드로 집계됩니다.
