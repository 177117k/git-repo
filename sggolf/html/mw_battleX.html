배틀X는 플레이 결과에 따라 배틀점수를 걸고 승부를 펼치는 모드입니다.

배틀유형

AI 배틀 AI를 상대로 플레이합니다.
동반자 배틀 동반자들끼리 개인 또는 팀으로 플레이합니다.

개인전 / 팀전 방식

개인전 각 개인의 성적을 바탕으로 승부 결정
팀전 팀 합산 성적을 바탕으로 승부 결정

배틀점수 규칙

  • 타당 배틀점수 :타당 100점, 상호 정산 방식
  • 부스트 옵션 활성 시, 3홀마다 20점씩 증가
    1홀~3홀타당 100점
    4홀~6홀타당 120점
    7홀~9홀타당 140점
    10홀~12홀타당 160점
    13홀~15홀타당 180점
    16홀~18홀타당 200점

배판 시스템

배판인 홀은 타당 획득 점수를 N배로 정산합니다.
(최대 9배 중첩)

  • 자동 배판 : 동점 시, 다음 홀 자동 적용
  • 수동 배판 : 사용 시, 해당 홀 적용
    • 플레이 중, 티샷 상황에서 배판 기능을 직접 사용할 수 있습니다.
    • 수동 배판 사용 제한 : 한 홀당 1회 / 라운드 중 총 2회

특별 점수

PAR3 홀 니어리스트 달성 시, 상대로부터 200점 획득
PAR5 홀 롱기스트 달성 시, 상대로부터 200점 획득
이글 상대로부터 200점 획득
알바트로스 이상 상대로부터 500점 획득

시즌 & 랭킹

  • 시즌 단위 : 3개월(분기)
  • 랭킹 집계 : 시즌 내 달성한 배틀점수 기준
    (기본 점수 이하의 점수는 집계에서 제외됩니다)
    * 기본 점수 : 10,000점

입장

  • 입장 조건 : 5,000점 이상 배틀점수 보유
    • 점수 부족 시, S포인트로 기본 점수 복구 가능

전적 관리

  • 승무패 전적은 계속 누적되며, 시즌별 초기화되지 않습니다.
    • S포인트로 초기화 가능
